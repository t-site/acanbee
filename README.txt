acanbee memory mungler

Akanbee means "nope" in japanese.

acanbee in a memory mungler written in C language.

it sometime allocate memory area , and zeroing.

it is work at sometime. just not a "hunter" or a "buster".

it is just small antimalware. no update data required.

For many machines include runs Linux , UNIX , and IoT computing things.
this repo includes acb.service for systemd based macines.

notice for IoT things , their randomness are poor , also use to rngd(8) or haveged(8).
